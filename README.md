### 一、系统定义
Zabbix Auto Api：开发并封装Zabbix Api，提升zabbix监控的自动化能力，提升运维人员的效率，可以将更多的时间用于自动化工具开发上来。
### 二、主要功能
封装zabbix api为私有云和CMDB提供监控创建、删除、分组、维护、责任人、自动屏幕等功能的RESTFUL接口。
